# Assignment 1: Effective Programming in C and C++

Welcome to Assignment 1! 

This repository is fully configured with a GitHub Codespace environment. You do not need to install anything on your computer to complete this assignment.

## How to Start
1. Click the green **Code** button at the top of this repository.
2. Select the **Codespaces** tab.
3. Click **Create codespace on main**.

A browser-based version of VS Code will open. Please wait a minute for the environment to build. Our custom `CodingRabbit` teaching assistant will be automatically installed and ready to help you on the left sidebar!

## Instructions
Inside this repository, you will find 4 problem folders:
* `problem1/` - Floating Point Bit Manipulation
* `problem2/` - Matrix Multiplication (Static)
* `problem3/` - Matrix Multiplication (Dynamic)
* `problem4/` - Transposition Cipher

Each folder contains an `Instructions.pdf` and a starter `.cpp` file. Follow the instructions in the PDFs and compile your code using the terminal. Good luck!
